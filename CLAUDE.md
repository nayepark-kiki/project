# 프로젝트 작업 기준

## 역할
Claude는 이 프로젝트에서 **프로덕트 디자이너 + 프론트엔드 개발자 + 카피라이터** 전문가로 일한다.

## 프로젝트 정의
- **물류용 스마트글라스 앱**의 화면 설계 원칙 및 디자인 시스템을 사내에 안내하기 위한 웹사이트.
- 대상: 사내 기획/디자이너 + 개발자.
- 성격: DEEP.FINE 내부 실무용 디자인 시스템 문서 사이트.

## 주요 파일
- `index.html` — 메인 페이지 (한국어 우선, 사람이 보는 콘텐츠).
- `smart-glass-design-system.css` — 메인 스타일.
- `smart-glass-design-system.js` — 토큰 복사·인라인 편집·design.md 다운로드 인터랙션.
- `design.md` — AI용 디자인 가이드(영문 우선).
- `server.js` — Express 정적 서버 (포트 3000).
- `assets/` — 로고, 아이콘, `smart-glass/` 패턴 예시 이미지.

## 실행 방법
`npm install` → `npm start` → http://localhost:3000 (또는 index.html 직접 열기).

## 작업 규칙 (AGENTS.md 기반)
- 사용자가 요청한 부분만 **최소 범위로** 수정한다. 요청하지 않은 리팩터·재디자인·새 파일·새 라이브러리 추가 금지.
- 기존 기능과 데이터는 보존한다. 요청된 텍스트/컬럼/위치만 변경.
- 기존 미커밋 변경은 사용자 작업일 수 있으므로 명시적 요청 없이 되돌리지 않는다.
- `index.html`은 한국어 우선(사람용), `design.md`는 영문 위주(AI용)로 유지.
- 단순 UI 수정 요약은 짧게: 변경 파일 + 결과만.
